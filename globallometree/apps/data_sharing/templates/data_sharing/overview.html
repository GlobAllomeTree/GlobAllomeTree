GlobAllomeTree Shared Data

GlobAllomeTree allows you to share and license your data in an easy way. It also lets you find data shared by others and request use of that data by agreeing to the terms and conditions.

Share Your Data

Sharing yor data is an easy process and involves the following steps.

  • Decide if use of your data will be restricted or not.
  • For restricted data you may use our license creation tool on this site to create your own license. It is also easy to choose a creative commons license for your data.
  • Upload your data in one of the formats that can be used by GlobAllomeTree (txt, json, or xml) or upload a pdf, excel, or other document that contains your data. GlobAllomeTree is currently accepting raw data, biomass expansion factors, allometric equations, and wood density data. The txt format should use a {TAB} delimiter and use UTF-16. This can be exported from excel using the UTF-16 Unicode Text (.txt) option, or from Open Office and other programs using the correct configuration.
  • For restricted data, you have the option to approve or deny requests from individual users directly on the GlobAllomeTree website. You will be sent an email from this site when a new request is made.

Discover and Use Shared Data

You can find and use shared data through the regular GlobAllomeTree search interface. Data that has an open license is already has the values available, while data that is subject to a license has some values hidden until you have an approved data sharing agreement with the data provider.
